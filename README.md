Visual Counter module
=====================

Basic Usage
-----------

    \usemodule[visualcounter]


    \definevisualcounter
      [....] % visualizer
      [....] % inherit from
      [counter=...,
       ...]

    \usevisualcounter{....}

Available Counters
------------------

- `scratchmarks` 
- `mayanumbers`
- `countdown`
- `markers`
- `progressbar`
- `pulseline`
